# Skill: 更新账号策略

## 触发条件
用户说以下内容时触发：
- 「更新账号策略」
- 「更新账号打法」
- 「更新能力画像」
- 「看看现在账号擅长什么」

## 执行步骤
1. 读取 `02-业务运营/业务规划/✅ 已完成的运营动作.md`
2. 读取 `01-内容生产/数据统计/内容数据表.md`
3. 更新 `02-业务运营/业务规划/账号运营策略.md`：
   - 平台角色
   - 指标阈值
   - 必做项
   - 停做项
   - 当前优先方向
4. 更新 `02-业务运营/业务规划/账号能力画像.md`：
   - 当前最擅长什么
   - 当前不擅长什么
   - 平台适配画像
   - 现阶段最该继续验证什么
5. 只根据已完成动作和数据结果反推，不要凭印象扩写

## 注意事项
- 如果 `✅ 已完成的运营动作.md` 没有新增事实，尽量不大改策略
- 口径冲突时，以 `内容数据表.md` 中已明确标注的指标为准
- 更新策略时，要明确说明哪些判断仍然只是待验证假设
