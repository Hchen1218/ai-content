# ai-content

ASCII 入口文件，方便在工具链和终端里快速定位这个仓库。

主入口：
- `运营OS.md`

长期维护的主层：
- `01-内容生产/`
- `02-业务运营/`
- `03-工具与效率/`
- `workflows/`
- `analysis/`
- `exports/`

约定：
- `workflows/` 只放仓库内业务工作流文档，不等同于 Codex 系统技能。
- `analysis/` 只保留可复用的结论型分析。
- `exports/` 只保留明确要分享或交付的最终导出物。
- 原始抓取、截图、帧图、音频、session、运行日志统一视为缓存，放在 `.cache/content-pipeline/`，不进入 Git。
