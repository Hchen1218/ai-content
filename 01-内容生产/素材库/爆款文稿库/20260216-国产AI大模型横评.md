## 标题策略

- **抖音封面标题**：AI工具大推荐 2026 5+纬度分析评分 90%的人都选错了工具
- **小红书标题**：千万别选错AI 工具！2026年超详细推荐合集

### 标题公式
**「反常识警告 + 年份 + 横评/推荐」**

关键要素：
1. 「90%的人选错了」→ 制造焦虑，触发点击
2. 封面点击率33.2%（全账号最高） → 标题极其有效
3. 小红书标题不同但也扑街 → 问题在分发而非标题

## 爆款原因分析

### 为什么抖音爆？

1. **封面点击率33.2%**：全账号最高，标题+封面组合极其有效
2. **平均播放时长33.8s**：长视频（5min+）能看33秒，说明内容承接力强
3. **5s完播率54.7%**：高于账号均值，钩子有效
4. **按场景推荐，不是排名**：差异化角度——「你是谁」决定「你该用谁」
5. **有明确结论**：每个产品给了明确的推荐/不推荐，不和稀泥
6. **评分表引导截图**：「暂停截图」= 收藏行为替代
7. **涨粉179**：全账号单条最高涨粉，系列预告（国外篇）创造期待

### 为什么小红书扑街？

- 5分钟长视频在小红书分发极弱
- 同类横评内容小红书竞争激烈，图文形式更适合
- 标题切口「千万别选错」不如抖音版有冲击力

## 可复用结构

### 内容结构模板：「N个产品横评」

```
1. 钩子（15s）：反常识开场（90%的人选错了）
2. 产品逐个拆解（每个30-90s）：
   - 核心优势（1-2个）
   - 明确短板
   - 适合谁（不是所有人）
   - 价格
3. 按场景推荐总结表（30s）：引导暂停截图
4. 评分总表（15s）：暂停截图保存
5. CTA：预告下一期 + 点赞收藏
```

### 关键成功要素清单
- [ ] 有明确结论（推荐/不推荐），不和稀泥
- [ ] 按「你是谁」而非「谁最好」推荐
- [ ] 有可暂停截图的总结表
- [ ] 系列预告创造期待（涨粉利器）
- [ ] 每个产品有1个记忆点（豆包=水桶机，智谱=买不到）
- [ ] 幽默点缀（千问点奶茶、百度不知道还活着）
