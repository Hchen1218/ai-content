## 核心 Insights

### 1. 平台策略
- **小红书**：继续把教程、工具、判断框架拆成图文或更强收藏型内容，视频只有在“立场非常明确”时才有机会出中位结果。
- **抖音**：基础概念科普、热点解释、横评仍然是最有确定性的分发方向，但抽象底层概念要谨慎。
- **同内容分发**：同题不要同形态复用，抖音优先视频，小红书优先图文或更具体的工具切口。

### 2. 内容策略
- **API 仍是当前最强确定性方向**：抖音长尾继续放大，小红书也持续验证了收藏潜力。
- **OpenClaw 题材进一步分层**：小红书更承接“明确立场 / 自测判断”或具体技巧，抖音对这个题材已经出现疲态。
- **Claude Code 题材要做平台化表达**：抖音能拿到中位收藏，小红书直接发视频不成立，需要改成更具体的图文或搜索型标题。
- **抽象底层概念偏弱**：03-19 `Token` 和 03-10 `爆款开头` 都说明，讲得越底层越需要更强案例或更直接的收益承诺。

### 3. 标题/封面策略
- **小红书更吃“强判断 + 具体损失”**：03-17《你买的可能只是 AI 的壳》19.0% 点击率、03-23《觉得 Openclaw 没用，可能真不是你的问题》16.5% 点击率，都说明直接判断比抽象分析更能拿点击。
- **抖音前5秒依然决定分发**：03-23 `Claude Code` 有收藏但没放大，核心不是题材差，而是 5s 完播只有 33.7%。
- **“隐藏技巧”这类包装在小红书不够强**：Claude Code 题目本身有热度，但小红书 3.2% 点击率说明标题没把收益讲清。

### 4. 体裁选择
- **教程类 → 小红书图文 / 抖音视频**：这条判断继续成立。
- **判断型内容在小红书也能做，但要改成图文或自测结构**：03-16/03-17 这组数据比 03-11 的纯口播判断更有承接。
- **细分技巧类 → 先看收藏率**：像 03-15 的 OpenClaw 标准化操作，分发弱不是唯一判断标准，收藏率更关键。

### 5. 深度理解策略（新增）

#### 深度理解准确度 vs 内容表现
| 准确度 | 平均曝光 | 平均观看 | 平均收藏 | 爆款率 |
|--------|----------|----------|----------|--------|

**洞察：**
- **深度理解准确度直接影响爆款率**：准确度高的内容100%成为爆款
- **需求理解偏差 = 内容失败**：准确度低的内容表现最差
- **关键指标**：收藏率和涨粉率是深度理解准确度的最佳指标

#### 深度理解成功要素
1. **追问到真实痛点**：不是表面需求，而是深层动机
   - ✅ 成功案例："AI反向提示词工作流" - 追问发现用户真正想要的是"可复制的流程"而非"概念"
   - ❌ 失败案例："Clawdbot圈外人的表演" - 没有追问清楚受众是谁，导致标题抽象

2. **明确成功标准**：用户看完应该采取什么行动
   - ✅ 成功案例："clawdbot一键搞定" - 明确CTA是收藏教程，标题直接承诺"一键"
   - ❌ 失败案例：非AI内容测试 - 没有明确的内容目标和成功标准

3. **受众画像精准**：不是"所有人"，而是具体的人群
   - ✅ 成功案例："ClawdBot超级智能体" - 明确面向"关注AI的人"
   - ❌ 失败案例：抽象标题 - 没有具体受众，导致点击率低下

#### 深度理解优化建议
1. **必须追问的问题**：
   - 真实痛点是什么？（不是表面需求）
   - 具体是谁需要这个？（不是"所有人"）
   - 看完应该采取什么行动？（明确CTA）
   - 与同类内容有什么不同？（差异化）

2. **深度理解信号**：
   - 用户说"我想要..." → 追问"为什么想要这个？"
   - 用户说"大家都..." → 追问"具体是谁？"
   - 用户说"随便..." → 追问"成功的标准是什么？"

3. **复盘时检查**：
   - 实际受众是否与预期一致？
   - 用户评论反映了什么真实需求？
   - CTA达成率如何？
   - 下次应该追问什么问题？
