# 选题方法论

> 如何找到好选题，如何评估选题潜力

## 选题来源

### 1. 热点追踪
- AI 工具官方公告
- 科技媒体报道
- 社交媒体讨论

### 2. 用户需求
- 评论区高频问题
- 私信咨询
- 搜索关键词

### 3. 自己实践
- 工具使用心得
- 效率提升方法
- 踩坑经验

### 4. 竞品分析
- 同类账号爆款
- 对标账号选题

## 选题评估标准

| 维度 | 高分 | 低分 |
|------|------|------|
| 需求 | 很多人有这个问题 | 小众需求 |
| 时效 | 当前热点 | 过时话题 |
| 差异 | 我有独特视角 | 大家都在做 |
| 执行 | 我能做好 | 超出能力 |

## 选题评分卡

评估新选题时，为每个维度打 1-5 分：

| 选题 | 需求 | 时效 | 差异 | 执行 | 总分 |
|------|------|------|------|------|------|
| | | | | | |

**决策规则**：
- 总分 ≥ 16：立即深化
- 总分 12-15：放入待观察
- 总分 < 12：暂时放弃
