# 选题方法论

> 如何找到好选题，如何评估选题潜力

## 选题来源

### 1. 热点追踪
- AI 工具官方公告
- 科技媒体报道
- 社交媒体讨论

### 2. 用户需求
- 评论区高频问题
- 私信咨询
- 搜索关键词

### 3. 自己实践
- 工具使用心得
- 效率提升方法
- 踩坑经验

### 4. 竞品分析
- 同类账号爆款
- 对标账号选题

## 选题评估标准

| 维度 | 高分 | 低分 |
|------|------|------|
| 需求 | 很多人有这个问题 | 小众需求 |
| 时效 | 当前热点 | 过时话题 |
| 差异 | 我有独特视角 | 大家都在做 |
| 执行 | 我能做好 | 超出能力 |

## 选题评分卡

评估新选题时，为每个维度打 1-5 分：

| 选题 | 需求 | 时效 | 差异 | 执行 | 总分 |
|------|------|------|------|------|------|
| | | | | | |

**决策规则**：
- 总分 ≥ 16：立即深化
- 总分 12-15：放入待观察
- 总分 < 12：暂时放弃

## 高分选题特征
- （根据数据统计持续更新）

### 已验证高分模式（基于15条历史数据）

| 模式 | 代表内容 | 数据验证 |
|------|---------|---------|
| **基础概念 + 生活化比喻** | API科普（水管比喻） | 抖音93K播放，小红书收藏率8.3% |
| **实用工具 + 可复制资源放评论区** | 3个AI提示词技巧 | 双平台超级爆款，抖音222K/小红书74K |
| **热点工具 + 48h内发布** | Seedance上手指南、ClawdBot | 抖音55K/3.5K |
| **横向对比 + 明确结论不和稀泥** | 国产AI大模型横评 | 抖音39K，封面点击率33.2%全账号最高 |
| **教程类 + 图文形式（小红书）** | clawdbot一键搞定 | 21K曝光，点击率12.9% |

### 高分选题共性
1. 有明确的「带走感」：提示词/工具链接/评分表可截图
2. 钩子在前3秒：数据震惊 or 反差场景
3. 与受众日常场景强相关
4. 有自己的真实使用数据（不是假设）

## 低分选题陷阱
- （根据数据统计持续更新）

### 已验证扑街模式

| 陷阱 | 代表内容 | 数据验证 |
|------|---------|---------|
| **非垂直内容（偏离AI领域）** | 贾乃亮K线、人生K线 | 小红书曝光627/2070，点赞个位数 |
| **热点发布时机晚3天以上** | Seedance 2.0观点视频（晚于教程4天） | 抖音692、小红书35 |
| **标题抽象无好奇缺口** | 「Clawdbot是圈外人的表演」 | 小红书点击率4.6%，观看314 |
| **纯分析视角，无体验感** | Clawdbot背后真相 | 抖音417、小红书314 |
| **标题过长/信息过载** | 全网疯抢一分钱奶茶 | 抖音628、小红书36 |

### 扑街共性
1. 观众看完不知道该干什么（无CTA）
2. 缺少「与我有关」感
3. 讲道理多于讲体验
4. 错过热点窗口期
