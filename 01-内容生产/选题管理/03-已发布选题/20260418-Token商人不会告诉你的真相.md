## 完整口播文案（补录）

最近你要是刷闲鱼、抖音、小红书，应该都看过一种很新的 AI 生意。叫做卖 token，一个月赚好几万，然后还有发帖子人教你怎么做这门生意，说这是“00 后通天路”，是“AI 时代普通人最好的机会”。但我想说，你看到的这些教程和截图，其实也是这门生意的一部分。

先说这门生意到底是什么。比如说像 Anthropic 旗下的 Claude 这种顶级大模型，确实好用，但因为地区、支付、账号这些门槛，很多人想用，却用不上。于是就出现了一批中间人。他们手里有 Claude 的账号、有 API、有 token，再转手卖给那些想用、但是不会弄，或者拿不到账号的人。

听上去很合理，对吧。你有钱，我有 token，你给我钱，我给你用。但问题就在这。你卖的其实不是 Claude 的能力，你卖的是“别人用不了 Claude”这件事。也就是说，这个生意本质上赚的不是 AI 的钱，赚的是一个访问缺口的钱。

而这种缺口型生意，前期非常暴利，因为会弄的人少，想用的人多，信息差又足够大。他们吃到的根本不是什么技术红利，吃到的是门槛红利。谁先拿到稳定账号，谁先打通支付和接口，谁就能先把这个缺口变成钱。

而这种缺口型生意都天生有一个很致命的问题：就是这个缺口不是你造出来的，所以你也根本守不住。如果明天出来一个同样好、价格只有一半的模型，还有人买你的吗？如果 Anthropic 明天把账号规则收紧了或者干脆放开了，大家都不能用或者都能用了，你的生意大概率都不在了。

赚快钱这件事本身没有错，谁不想赚快钱，但你至少要知道这个东西为什么能赚钱。你赚到的，到底是你自己创造出来的价值，还是一个随时会被堵上的缺口。

而这个缺口，现在已经在缩小了。从 4 月 4 日开始，Anthropic 就已经不再允许用户把 Claude 的订阅额度，用在 OpenClaw、小龙虾这类第三方 agent 上。连 OpenClaw 创始人的号都被封了。而就在昨天，Anthropic 又进一步上线了身份验证机制。

有些场景下，用户已经会被要求提交身份证件，甚至要求做自拍验证。这说明，这门生意最核心的资产，也就是账号和接口，正在被 Anthropic 重新抓回自己手里。

而当做这一行的人越来越多，利润空间被压榨的时候，真正更赚钱的东西，就不再是 token 本身了。他们会开始卖一个更轻、更快、利润更高的东西：不是卖 token，而是贩卖“卖 token 的机会”。也就是收代理、卖账号、卖话术、卖几千块的入局资格。

表面上看，你是在花钱让他教你做一门 AI 生意。实际上他在卖的，是“你也可以靠 AI 赚钱”的想象。而当你交学费的那一刻，钱就已经不是从 token 本身来的了，而是从下一批想靠卖 token 赚钱的人身上来。

这时候发生的，就是利润迁移。利润已经不在产品本身了，而是迁到了教程、代理费、学费、入局资格上。再往后，连代理都不好卖了，就会继续一层一层往下收割。每往下走一层，就离真实需求远一步。

所以 token 中转不是天生骗局。它一开始就是一个利用访问缺口赚钱的生意。但当一门生意不靠挖金子赚钱，开始靠卖铲子赚钱，那你看到的就已经不是一门正常生意了。

真正能长期做的生意，护城河都是你自己建的，可以是技术，是品味，是信任，是规则的漏洞，但不能是下一批人的学费。说到底，你以为你在学怎么卖 token，其实很多时候，你学的是怎么成为被收割的下一环。
