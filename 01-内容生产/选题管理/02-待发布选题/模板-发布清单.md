# 发布检查清单

> 发布前逐项检查，避免低级错误

## 内容检查
- [ ] 标题是否吸引人？
- [ ] 封面/首图是否清晰？
- [ ] 开头前5秒是否有钩子？
- [ ] 内容逻辑是否清晰？
- [ ] 结尾是否有 CTA（关注/点赞/评论）？

## 格式检查
- [ ] 文字无错别字？
- [ ] 图片/视频清晰度足够？
- [ ] 字幕（如有）无误？

## 平台适配
- [ ] 小红书：标题字数 < 20？
- [ ] 小红书：添加了相关话题标签？
- [ ] 抖音：视频时长适中（1-3分钟）？
- [ ] 即刻：配图尺寸正确？
