## 完整口播文案（定稿）

【钩子 ~10s】
我有一个朋友，专门搞开发的。
有一天跑来问我：你能帮我配一下API吗？
我说可以，但要收钱。
他说行。
一个写代码的，不会配自己的API Key。
这玩意其实一点都不难。

【定义 ~15s】
API，你把它理解成一根水管就行。
AI公司是自来水厂，它生产水。
API Key就是你家那根接到户的水管。
水接进来之后，你想接洗衣机、洗碗机还是花洒，你自己决定。

【真实数据建立信任 ~15s】
我买了Google的AI套餐，
每周大概用掉几千万的token。
它给了我一个API Key，就这么一串字符。
然后我拿这一串字符做了几件事。

【场景1 ~20s】
第一件——
Claude Code，最近很火的AI编程工具。
我把我的Google API接进去，
后台跑的是Google自己的大模型，
比直接订阅原生版本，便宜很多。

【场景2 ~20s】
第二件——
OpenClaw，就是那个两天涨了20万Star的龙虾机器人。
它能帮你在飞书、Discord、微信同时干活，
但它需要你自己接入大模型才能用。
我把我的Google API Key填进去，
它直接活了。

【场景3 ~20s】
第三件——
沉浸式翻译，很多人在用的那个浏览器翻译插件。
它有免费版，但翻译质量一般。
接上你自己的API Key之后，
它调用的就是你买的大模型，
翻译质量直接上一个档次，而且几乎不花钱。

【核心总结 ~10s】
一根水管，接了三台电器。
一个编程工具，一个AI助手，一个翻译插件。
这就是API的逻辑——买一次水，用在所有地方。

【反差+紧迫 ~20s】
现在网上有大量免费的开源工具，
但它们需要你自己接水管。
不会用API，
你就只能用别人帮你套好壳的版本，还要多付一层钱。
懂API的人，用同样的预算，能用到的工具是你的好几倍。

【行动号召 ~15s】
你现在可以做一件事：
去Google AI Studio，或者Kimi、火山引擎——
它们都有免费额度。
找到"API Key"，生成一个，先放在那里。
下次遇到需要接入大模型的工具，你就直接有了。
评论区告诉我，你第一个想接的是什么工具。
