# Skill: 记录选题

## 触发条件
用户说以下内容时触发：
- 「记录选题」
- 「记录一个想法」
- 「有个选题想法」
- 「想做一个选题关于...」

## 执行步骤
1. 提取用户的选题想法
2. 确认来源（热点/用户提问/自己实践/其他）
3. 追加到 `01-内容生产/选题管理/00-选题记录.md`
4. 使用表格格式，自动填充日期
5. 潜力默认为 ⭐⭐⭐

## 输出格式
确认信息：
- 已记录选题：[选题名]
- 来源：[来源]
- 下一步：可以说「深化选题」来展开这个想法

## 示例对话

**用户**：记录选题，想做一期 Claude Code 的深度测评

**AI**：已记录选题！
- 选题：Claude Code 深度测评
- 来源：自己实践
- 潜力：⭐⭐⭐
- 已添加到选题记录表

要现在深化这个选题吗？

## 注意事项
- 只记录，不展开
- 保持简洁，一句话描述
- 如果用户没说来源，默认填「灵感」
