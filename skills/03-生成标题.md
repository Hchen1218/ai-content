# Skill: 生成标题

## 触发条件
用户说以下内容时触发：
- 「生成标题」
- 「小红书标题」
- 「帮我想个标题」
- 「标题怎么写」

## 执行步骤
1. 理解内容主题和核心卖点
2. 读取 `方法论/小红书标题方法论.md`
3. 应用多个标题公式生成选项
4. 为每个标题解释逻辑
5. 推荐最优选项

## 输出格式
```
## 标题选项

### 选项 1（推荐）
[标题]
- 公式：[使用的公式]
- 逻辑：[为什么这样写]

### 选项 2
[标题]
- 公式：[使用的公式]
- 逻辑：[为什么这样写]

### 选项 3
[标题]
- 公式：[使用的公式]
- 逻辑：[为什么这样写]

## 推荐理由
[为什么推荐选项1]
```

## 示例对话

**用户**：帮我生成标题，内容是关于用 Claude 写小红书文案

**AI**：
## 标题选项

### 选项 1（推荐）
**3步教你用Claude写出爆款小红书文案**
- 公式：数字+结果
- 逻辑：数字降低阅读门槛，"爆款"是用户想要的结果

### 选项 2
**还在为写文案发愁？Claude 帮你10分钟搞定**
- 公式：痛点+解决方案
- 逻辑：直击用户痛点，给出明确解决方案

### 选项 3
**我用 Claude 写文案，粉丝问我请了什么写手**
- 公式：故事/结果导向
- 逻辑：用故事引发好奇，暗示效果很好

## 推荐理由
选项1最直接，数字型标题在小红书点击率通常更高，同时"爆款"这个词符合用户追求。

## 注意事项
- 小红书标题控制在 20 字以内
- 优先使用数据统计中验证过的高点击率关键词
- 避免使用数据统计中记录的低点击率陷阱
