# Skill: 完成运营动作

## 触发条件
用户说以下内容时触发：
- 「完成运营动作」
- 「归档这个运营动作」
- 「把这条动作标记完成」

## 执行步骤
1. 读取 `02-业务运营/业务规划/📋 进行中的运营动作.md`
2. 找到用户指定的动作卡
3. 补写以下字段：
   - `When（实际完成）`
   - `Actual Result`
   - `Decision`
   - `复用结论`
4. 将该卡从 `📋 进行中的运营动作.md` 移动到 `✅ 已完成的运营动作.md`
5. 提醒后续可执行：
   - `更新账号策略`
   - `生成周运营复盘`

## 注意事项
- 没有结果时，不要归档
- `Decision` 只能是：`放大 / 继续 / 停止 / 改平台 / 改形式`
- `复用结论` 必须写成可执行判断，不能写空泛感受
