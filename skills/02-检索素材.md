# Skill: 检索素材

## 触发条件
用户说以下内容时触发：
- 「检索素材」
- 「找相关素材」
- 「有没有类似的内容」

**隐式触发**（自动执行）：
- 在深化选题时
- 在创作任何内容之前

## 执行步骤
1. 确定搜索关键词（从用户输入或选题提取）
2. **按顺序搜索**以下目录：
   - `素材库/AI工具库/` - 相关工具资料
   - `素材库/观点库/` - 相关观点和思考
   - `素材库/金句库/` - 可复用的表达
   - `素材库/爆款文稿库/` - 类似主题的爆款
3. 汇总搜索结果
4. 给出复用建议

## 输出格式
```
## 素材检索结果

### 相关工具
- [工具名] - [简述]

### 相关观点
- [观点] - [来源]

### 可用金句
- [金句] - [适用场景]

### 类似爆款
- [标题] - [可复用点]

## 复用建议
[基于检索结果，给出具体的复用建议]
```

## 示例对话

**用户**：检索素材，关于 Claude 的使用技巧

**AI**：
## 素材检索结果

### 相关工具
- Claude.md - 已收录工具卡片，包含核心功能和使用场景

### 相关观点
- "Claude 的 Artifact 功能改变了 AI 协作方式" - 来源：自己思考

### 可用金句
- "不是 AI 取代人，而是会用 AI 的人取代不会用的人" - 适用场景：开头/结尾

### 类似爆款
- 暂无

## 复用建议
1. 可以复用 Claude.md 中的功能介绍作为基础
2. 开头可以用金句库中的表达
3. 建议创作完成后归档到爆款文稿库

## 注意事项
- 【重要】这是最核心的 Skill，必须在创作前执行
- 没有找到相关素材也要明确告知
- 找到相关素材时，具体说明如何复用
